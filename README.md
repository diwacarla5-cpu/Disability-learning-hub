# Learning Disabilities Learning System

## Overview

The Learning Disabilities Learning System is a web-based educational platform developed using PHP, MySQL, HTML, CSS, and JavaScript. It is designed to raise awareness and provide educational resources about learning disabilities through interactive lessons and quizzes.

---

## Features

- User Registration and Login
- Learning Modules
- Interactive Quiz
- Progress Tracking
- User Profile
- Admin Dashboard
- User Management
- Lesson Management
- Quiz Management
- Responsive Design
- MySQL Database

## Installation

1. Install XAMPP.
2. Copy the project folder to xampp/htdocs/.
3. Start Apache and MySQL.
4. Open phpMyAdmin.
5. Create a database named learning_disabilities.
6. Import the database.sql file.
7. Open your browser and go to:

http://localhost/learning-disabilities
